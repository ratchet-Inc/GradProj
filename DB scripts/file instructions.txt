The python scripts found in this directory are to be utilized for creating a usable valid database for the recommendation system.




The database movie list can be sourced from: https://www.imdb.com/interfaces/




The following to commands must be ran to convert the tsv files to usable sql files for queries:-

python TSVtoMySQL.py -tf data(basics).tsv -tn titles -cl vc(128) nn pk, vc(64) nn, vc(512) nn, vc(512) nn, int(16) nn, vc(32) nn, vc(32) nn, int(32) nn, vc(128) nn -of COMP3901 -db moviesDB -ndb true -flt true, movie

python TSVtoMySQL.py -tf data(akas).tsv -tn titleDetails -cl vc(64) nn fk, int(32) nn, vc(384) nn, vc(128) nn, vc(32) nn, vc(128) nn, vc(64) nn, int(8) nn -of COMP3901 -db moviesDB -ndb false -flt false, <refFileName>, <refTable>, <refColumn>

python TSVtoMySQL.py -tf data(names).tsv -tn crewNames -cl vc(16) nn pk, vc(64) nn, int(8) nn, int(8) nn, vc(256) nn, vc(1024) -of COMP3901 -db moviesDB -ndb false -flt false, none


Following the queries abow, it is time ti insert into the databse via the scripts.
Run the following command line instruction to executed the script:

mysql -u root -p<password> > <filePath>/COMP3910(<fileName>).sql

* where '<>' is located, specific target information is required
