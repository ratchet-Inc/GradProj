The python scripts found in this directory are to be utilized for creating a usable valid database for the recommendation system.



The database movie list can be sourced from: https://www.imdb.com/interfaces/

The following to commands must be ran to convert the tsv files to usable sql files for queries:-

python TSVtoMySQL.py -tf data(basics).tsv -tn titles -cl vc(64) nn pk, vc(32) nn, vc(128) nn, vc(256) nn, int(8) nn, vc(16) nn, vc(16) nn, int(16) nn, vc(64) nn -of COMP3901 -db moviesDB -ndb true

python TSVtoMySQL.py -tf data(akas).tsv -tn titlesDetails -cl vc(64) nn fk, int(32) nn, vc(256) nn, vc(128) nn, vc(32) nn, vc(128) nn, vc(64) nn, int(8) nn -of COMP3901 -db moviesDB -ndb false
